## Spring을 이용한 쇼핑몰 웹사이트
Spring Boot를 공부하며, 여러 기능을 담은 쇼핑몰 웹사이트를 만들고 배포해보았습니다.
배포 주소 : http://yoonyoulcodingapplespring1-env.eba-bpzregrh.ap-northeast-2.elasticbeanstalk.com/

### 사용한 언어
- Java
- MYSQL

### 사용한 프레임워크
- spring boot

### 사용한 Database
- Microsoft Azure

### 배포 환경
- AWS Elastic Beanstalk

### 구현한 기능
- JWT 로그인
